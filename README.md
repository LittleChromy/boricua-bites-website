## 🚀 Getting Started

### View the Website
The website is built with:
- **HTML5** - Semantic structure
- **CSS3** - Beautiful styling with Puerto Rican colors (Blue, Red, Gold, White)
- **JavaScript** - Interactive menus and form handling

### File Structure
```
boricua-bites-website/
├── index.html      # Main website
├── styles.css      # Styling
├── script.js       # Interactivity
└── README.md       # This file
```

### Deploy with GitHub Pages
1. Go to your repository Settings
2. Scroll to "GitHub Pages"
3. Select "main" branch as the source
4. Your site will be live at: `https://littlechrony.github.io/boricua-bites-website/`

## 📋 How the Order Form Works

1. **Fill in Customer Info** - Name, phone, email
2. **Select Order Type** - Pickup or Delivery
3. **Add Items** - Choose from the menu, set quantities
4. **Special Instructions** - Any dietary preferences or allergies
5. **Choose Payment Method** - Cash or Card
6. **Submit Order** - Confirmation message appears

### ⚠️ Important Notes
The order form currently:
- ✅ Collects all customer information
- ✅ Validates form entries
- ✅ Displays order confirmation
- ❌ Does NOT process payments (requires integration with Stripe, Square, or PayPal)
- ❌ Does NOT send emails automatically (requires backend setup)

**To make orders fully operational, you'll need to:**
1. Set up a payment processor (Stripe, Square, PayPal)
2. Create a backend to handle order submissions (or use a service like Formspree, Netlify Forms)
3. Integrate email notifications
